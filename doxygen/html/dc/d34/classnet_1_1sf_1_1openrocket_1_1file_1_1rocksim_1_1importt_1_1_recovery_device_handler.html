Detailed Description

A handler specific to streamers and parachutes. This is done because Rocksim allows any type of material to be used as a recovery device, which causes oddities with respect to densities. Density computation is overridden here to try to correctly compute a material's density in OpenRocket units.

Parameters
<C>either a Streamer or Parachute

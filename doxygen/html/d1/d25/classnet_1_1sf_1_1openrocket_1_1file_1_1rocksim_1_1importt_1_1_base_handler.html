§ computeDensity()

double net.sf.openrocket.file.rocksim.importt.BaseHandler< C extends RocketComponent >.computeDensity ( RocksimDensityType  type,
double  rawDensity 
)
protected

Compute the density. Rocksim does strange things with densities. For some streamer material it's in cubic, rather than square, units. In those cases it needs to be converted to an appropriate SURFACE material density. Some G10 fiberglass materials are in cubic units, other G10 fiberglass is in square units. And due to a Rocksim bug, some densities are 0 when they clearly should not be.

This may be overridden for specific component density computations.

Parameters
typethe rocksim density
rawDensitythe density as specified in the Rocksim design file
Returns
a value in OpenRocket SURFACE density units

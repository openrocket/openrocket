Detailed Description

This class is the main entry point for Rocksim design file imported to OpenRocket. Currently only Rocksim v9 file formats are supported, although it is possible that v8 formats will work for most components.

In the cases of v9 components that exist in Rocksim but have no corollary in OpenRocket a message is added to a warning set and presented to the user. In effect, this loading is a 'best-effort' mapping and is not meant to be an exact representation of any possible Rocksim design in an OpenRocket format.

Rocksim simulations are not imported.

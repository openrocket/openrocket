Detailed Description

This class handles Rocksim 'SubAssembly' elements. They are similar to 'AttachedParts' (which is why this class is subclassed from AttachedPartsHandler) with some key differences. In Rocksim, AttachedParts elements can contain SubAssembly elements, which can in turn contain AttachedParts elements. To represent them in OR, SubAssembly elements are treated as children of the stage - much like a nose cone or external body tube.

Called when an opening tag of a contained element is encountered. Returns the handler that will handle the elements within that element, or null if the element and all of its contents is to be ignored.

Note that this method may also return this, in which case this handler will also handle the subelement.

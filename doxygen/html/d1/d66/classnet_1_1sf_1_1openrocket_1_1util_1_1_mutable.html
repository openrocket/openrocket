Detailed Description

A utility class helping an object to be made immutable after a certain point of time. An object should contain an instance of Mutable and an immute() method which calls immute(). Additionally, every method that changes the state of the object should call check() before modification.

This class also provides a stack trace of the position where the object was made immutable to help in debugging modifications of immuted objects.

Detailed Description

This class is a Sax element handler for Rocksim version 9 design files. It parses the Rocksim file (typically a .rkt extension) and creates corresponding OpenRocket components. This is a best effort approach and may not be an exact replica.

Limitations: Rocksim flight simulations are not imported; tube fins are not supported; Rocksim 'pods' are not supported.

Called when an opening tag of a contained element is encountered. Returns the handler that will handle the elements within that element, or null if the element and all of its contents is to be ignored.

Note that this method may also return this, in which case this handler will also handle the subelement.

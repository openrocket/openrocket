Detailed Description

A cyclic buffer with a fixed size. When more data is inserted, the newest data will overwrite the oldest data.

Though this class implements the Queue interface, it specifically breaks the contract by overwriting (removing) data without specific removal. It also currently does not support removing arbitrary elements from the set.

The methods in this class are synchronized for concurrent modification. However, iterating over the set is not thread-safe. To obtain a snapshot of the state of the buffer, use asList().

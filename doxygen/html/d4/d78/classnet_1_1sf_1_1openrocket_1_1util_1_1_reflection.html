static Error net.sf.openrocket.util.Reflection.handleWrappedException ( Exception  e)
static

Handles an InvocationTargetException gracefully. If the cause is an unchecked exception it is thrown, otherwise it is encapsulated in a BugException.

This method has a return type of Error in order to allow writing code like: 

throw Reflection.handleInvocationTargetException(e)

This allows the compiler verifying that the call will never succeed correctly and ending that branch of execution.

Parameters
ethe InvocationTargetException that occurred (not null).
Returns
never returns normally.

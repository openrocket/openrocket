123 OpenRocket 13.05 (2013-05-04)
124 -----------------------------
125 
126 New Features
127 
128  * Added support for decals on rockets. Added two new 3d views - 3d finished and 3d unfinshed. Added support to export and
129  apply decals (images) to rocket components. Added ability to launch external graphics editor to edit decals and monitor
130  file for writes to update the rocket view.
131  * Added simulation of tumble recovery based on experimentation done by Sampo Niskanen. This is particularly useful
132  for low power staged flights.
133  * Extended "motor configuration" concept to cover more properties. The concept was renamed "Flight Configuration" and
134  allows the user to override stage separation, recovery deployment, and motor ignition per flight configuration.
135  These configurations are stored in the ork file with each component. If no override is specified then the user specified
136  default is used (set in the component edit dialog). The flight configuration dialog was reworked to make it more
137  usable. The user selects the configuration in a drop down with buttons for "add", "delete" and "rename". After selecting
138  the configuration, the tabbed area below allows the user to change the configuration for this simulation.
139  * Allow simulation of stages without a motor. Users in the past have attempted to simulate separate recovery of payload
140  and booster by tricking OpenRocket using a dummy motor with trivial thrust curve. Now the user does not need to do this.
141  There is an example of this provided in TARC Payloader.ork and Boosted Dart.ork
142  * Simulate recovery of boosters. The simulation engine will create new FlightDataBranches for each stage after
143  separation. The data for the boosters begins at t=separation. The simulation plot allows the user to select
144  which stage's data to show (or all).
145  * Modified the zoom and pan controls in the simulation plot. Added zoom in, out, and reset buttons. Added
146  scroll with mouse wheel. If the alt key is used with either of these, only the domain is zoomed. Richard
147  contributed a more logical mouse controlled zoom - right click and drag will zoom (either domain, range or both).

177 OpenRocket 12.03 (2012-03-17)
178 ------------------------------
179 
180 In this release the version numbering scheme has been changed to be
181 "YY.MM" indicating the year and month of the release.
182 
183 Enhancements in the desktop version include saving designs in RKT
184 format thanks to Doug Pedrick, freeform fin set import from images by
185 Jason Blood, configurable stage separation events, guided help tours
186 and displaying the computed motor designation class. The application
187 has also been translated to Italian by Mauro Biasutti and Russian by
188 the Sky Dart Team.
189 
190 This also marks the first release for Android devices. In this first
191 release you can open files and examine existing simulations, stability
192 data and motor files. The Android port is thanks to work by Kevin
193 Ruland.

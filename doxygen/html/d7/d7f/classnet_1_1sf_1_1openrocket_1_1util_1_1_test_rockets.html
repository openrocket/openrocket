§ makeTestRocket()

Rocket net.sf.openrocket.util.TestRockets.makeTestRocket ( )

Create a new test rocket based on the value 'key'. The rocket utilizes most of the properties and features available. The same key always returns the same rocket, but different key values produce slightly different rockets. A key value of null generates a rocket using a random key.

The rocket created by this method is not fly-worthy. It is also NOT guaranteed that later versions would produce exactly the same rocket!

Returns
a rocket design.

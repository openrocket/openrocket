§ toCamelCase()

static String net.sf.openrocket.preset.loader.RocksimComponentFileLoader.toCamelCase ( String  target)
staticprotected

Convert all words in a given string to Camel Case (first letter capitalized). Words are assumed to be separated by a space. Note: this is done because some manufacturers define their material name in Camel Case but the component part references the material in lower case. That causes matching/lookup issues that's easiest handled this way (rather than converting everything to lower case.

Parameters
targetthe target string to be operated upon
Returns
target, with the first letter of each word in uppercase

Detailed Description

This is the core Swing representation of a fin marking guide. It can handle multiple fin sets on the same or different body tubes. One marking guide will be created for any body tube that has a finset. If a tube has multiple finsets, then they are combined onto one marking guide. It also includes launch lug marking line(s) if lugs are present. If (and only if) a launch lug exists, then the word 'Front' is affixed to the leading edge of the guide to give orientation.

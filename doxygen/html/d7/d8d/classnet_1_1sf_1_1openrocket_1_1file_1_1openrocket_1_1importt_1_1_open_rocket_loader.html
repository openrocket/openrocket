Detailed Description

Class that loads a rocket definition from an OpenRocket rocket file.

This class uses SAX to read the XML file format. The #loadFromStream(InputStream) method simply sets the system up and starts the parsing, while the actual logic is in the private inner class OpenRocketHandler.

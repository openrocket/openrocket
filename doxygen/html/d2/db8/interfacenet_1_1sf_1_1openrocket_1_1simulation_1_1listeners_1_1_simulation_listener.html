Called when ending a simulation. This is called either when the simulation ends normally (due to an end simulation event) or when a SimulationException is thrown.

This method cannot throw a SimulationException, since the simulation is already being ended.

§ isSystemListener()

boolean net.sf.openrocket.simulation.listeners.SimulationListener.isSystemListener ( )

Return whether this is a system listener. System listeners are used internally for various purposes by OpenRocket. User-written listeners should always return false.

System listeners do not cause warnings to be added to the simulation results when they affect the simulation.

Returns
whether this is a system listener

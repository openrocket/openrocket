Detailed Description

A list of listeners of a specific type. This class contains various utility, safety and debugging methods for handling listeners.

Note that unlike normal listener implementations, this list does NOT allow the exact same listener (equality using ==) twice. While adding a listener twice to a event source would in principle be valid, in practice it's most likely a bug. For example the Swing implementation Sun JRE contains such bugs.

Iterator<T> net.sf.openrocket.util.ListenerList< T >.iterator ( )

Return an iterator that iterates of the listeners. This iterator is backed by a copy of the iterator list, so addListener(Object) and removeListener(Object) may be called while iterating the list without effect on the iteration. The returned iterator does not support the Iterator#remove() method.

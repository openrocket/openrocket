Detailed Description

Preliminary computation of tube fin aerodynamics.

Uses a complete clone of FinSetCalc modelling each tube fin as 3 individual fins. It does not correctly account for fin & tube fin interference.

Changes to BarrowmanCalculator's calculateFrictionDrag are also probably required.

Detailed Description

A group of units (eg. length, mass etc.). Contains a list of different units of a same quantity.

§ fromString()

double net.sf.openrocket.unit.UnitGroup.fromString ( String  str)

Converts a string into an SI value. If the string has one of the units in this group appended to it, that unit will be used in conversion. Otherwise the default unit will be used. If an unknown unit is specified or the value does not parse with Double.parseDouble then a NumberFormatException is thrown.

This method is applicable only for simple units without e.g. powers.

Parameters
strthe string to parse.
Returns
the SI value.
Exceptions
NumberFormatExceptionif the string cannot be parsed.

Detailed Description

GuiModule is the Guice Module for the OpenRocket Swing application.

Because the swing application does not fully utilize injection of dependencies, proper use of this module requires a little more work on the part of the startup code.

GuiModule module = new GuiModule(); Application.setInjector( Guice.createInjector(guiModule)); module.startLoading();

§ convert()

double net.sf.openrocket.gui.print.PrintUnit.convert ( double  sourceLength,
PrintUnit  sourceUnit 
)

Convert the given length in the given unit to this unit. Conversions from finer to coarser granularities truncate, so may lose precision.

For example, to convert 10 inches to point, use: PrintUnit.POINTS.convert(10L, PrintUnit.INCHES)

Parameters
sourceLengththe length in the given sourceUnit
sourceUnitthe unit of the sourceDuration argument
Returns
the converted length in this unit, or Long.MIN_VALUE if conversion would negatively overflow, or Long.MAX_VALUE if it would positively overflow.

§ POINTS_PER_INCH

static final int net.sf.openrocket.gui.print.PrintUnit.POINTS_PER_INCH = 72
static

PPI is Postscript Point and is a standard of 72. Java2D also uses this internally as a pixel-per-inch, so pixels and points are for the most part interchangeable (unless the defaults are changed), which makes translating between the screen and a print job easier.

Not to be confused with Dots-Per-Inch, which is printer and print mode dependent.

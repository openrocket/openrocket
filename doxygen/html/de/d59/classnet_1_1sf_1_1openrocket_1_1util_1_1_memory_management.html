Detailed Description

A class that performs certain memory-management operations for debugging purposes. For example, complex objects that are being discarded and that should be garbage-collectable (such as dialog windows) should be registered for monitoring by calling collectable(Object). This will allow monitoring whether the object really is garbage-collected or whether it is retained in memory due to a memory leak. Only complex objects should be registered due to the overhead of the monitoring.

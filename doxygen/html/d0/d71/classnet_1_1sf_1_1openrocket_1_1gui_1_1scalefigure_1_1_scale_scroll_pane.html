Detailed Description

A scroll pane that holds a ScaleFigure and includes rulers that show natural units. The figure can be moved by dragging on the figure.

This class implements both MouseListener and MouseMotionListener. If subclasses require extra functionality (e.g. checking for clicks) then these methods may be overridden, and only unhandled events passed to this class.

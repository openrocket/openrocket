§ AsynchronousDatabaseLoader()

net.sf.openrocket.database.AsynchronousDatabaseLoader.AsynchronousDatabaseLoader ( long  startupDelay)

Sole constructor.

The startupDelay parameter defines a time to delay after calling startLoading before actually starting the loading. This allows other actions such as GUI opening to proceed faster. The delay can be cancelled by calling setInUse().

Parameters
startupDelaynumber of milliseconds to delay before starting actual loading.

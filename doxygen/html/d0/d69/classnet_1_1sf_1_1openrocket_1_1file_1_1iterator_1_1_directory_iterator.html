Detailed Description

A DirectoryIterator that scans for files within a directory in the file system matching a FileFilter. The scan is optionally recursive.

Return a DirectoryIterator for a directory that can be located either within the containing JAR file, in the classpath or in the current directory (searched in this order). The first place that contains matching files will be iterated through.

Parameters
directorythe directory to search for.
filterthe filter for matching files in the directory.
Returns
a DirectoryIterator for iterating through the files in the directory, or null if no directory containing matching files can be found.

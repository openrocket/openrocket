Detailed Description

Model suitable for JSpinner. Note: Previously used using JSpinner.NumberEditor and extended SpinnerNumberModel to be compatible with the NumberEditor, but only has the necessary methods defined. This is still the design, but now extends AbstractSpinnerModel to allow other characters to be entered so that fractional units and expressions can be used.

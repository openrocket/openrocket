§ getUpdateInfo()

UpdateInfo net.sf.openrocket.communication.UpdateInfoRetriever.getUpdateInfo ( )

Retrieve the result of the background update info fetcher. This method returns the result of the previous call to start(). It must be called before calling this method.

This method will return null if the info fetcher is still running or if it encountered a problem in communicating with the server. The difference can be checked using isRunning().

Returns
the update result, or null if the fetching is still in progress or an error occurred while communicating with the server.
Exceptions
IllegalStateExceptionif start() has not been called.

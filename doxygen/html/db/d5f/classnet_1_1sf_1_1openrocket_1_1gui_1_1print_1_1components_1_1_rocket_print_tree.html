§ setCheckBoxSelectionModel()

void net.sf.openrocket.gui.print.components.RocketPrintTree.setCheckBoxSelectionModel ( TreeSelectionModel  checkBoxSelectionModel)

This tree needs to have access both to the normal selection model (for the textual row) which is managed by the superclass, as well as the selection model for the check boxes. This mutator method allows an external class to set the model back onto this class. Because of some unfortunate circular dependencies this cannot be set at construction.

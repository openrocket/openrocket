§ getManufacturer()

static Manufacturer net.sf.openrocket.motor.Manufacturer.getManufacturer ( String  name)
static

Returns a manufacturer for the given name. The manufacturer is searched for within the manufacturers and if a match is found the corresponding object is returned. If not, a new manufacturer is returned with display and simple name the name specified. Subsequent requests for the same (or corresponding) manufacturer name will return the same object.

Parameters
namethe manufacturer name to search for.
Returns
the Manufacturer object corresponding the name.

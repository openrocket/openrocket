Detailed Description

An element handler that does not allow any sub-elements. If any are encountered a warning is generated and they are ignored.

Called when an opening tag of a contained element is encountered. Returns the handler that will handle the elements within that element, or null if the element and all of its contents is to be ignored.

Note that this method may also return this, in which case this handler will also handle the subelement.

Parameters
elementthe element name.

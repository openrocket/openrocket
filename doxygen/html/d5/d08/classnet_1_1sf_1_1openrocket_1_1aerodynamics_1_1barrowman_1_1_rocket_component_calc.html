abstract

Calculate the non-axial forces produced by the component (normal and side forces, pitch, yaw and roll moments and CP position). The values are stored in the AerodynamicForces object. Additionally the value of CNa is computed and stored if possible without large amount of extra calculation, otherwise NaN is stored. The CP coordinate is stored in local coordinates and moments are computed around the local origin.

Parameters
conditionsthe flight conditions.
forcesthe object in which to store the values.
warningsset in which to store possible warnings.

Detailed Description

An implementation of a ParallelFunctionCache that evaluates function values in parallel and caches them. This allows pre-calculating possibly required function values beforehand. If values are not required after all, the computation can be aborted assuming the function evaluation supports it.

Note that while this class handles threads and abstracts background execution, the public methods themselves are NOT thread-safe and should be called from only one thread at a time.

Detailed Description

Use this class to build an immutable Appearance object in a friendly way. Set the various values one at a time with the setter methods and then call getAppearance(). Each call to getAppearance will return a new appearance.

You can use this class repeatedly and resetToDefaults() in between, or create a new one every time.

Return the motor to inject to the rocket design when loading a file. This may also return dysfunctional motor placeholders that will later be replaced with the appropriate motor.

Any parameter that is null/NaN should be ignored when finding the motors. The method can also add appropriate warnings to the provided warning set in case there are problems selecting the motor.

Returns
the motor to use in the design, or null for no motor.

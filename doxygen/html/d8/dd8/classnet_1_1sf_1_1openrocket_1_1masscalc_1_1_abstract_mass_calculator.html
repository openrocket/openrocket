§ checkCache()

final void net.sf.openrocket.masscalc.AbstractMassCalculator.checkCache ( Configuration  configuration)
protected

Check the current cache consistency. This method must be called by all methods that may use any cached data before any other operations are performed. If the rocket has changed since the previous call to checkCache(), then voidMassCache() is called.

This method performs the checking based on the rocket's modification IDs, so that these method may be called from listeners of the rocket itself.

Parameters
configurationthe configuration of the current call

§ voidMassCache()

void net.sf.openrocket.masscalc.AbstractMassCalculator.voidMassCache ( )
protected

Void cached mass data. This method is called whenever a change occurs in the rocket structure that affects the mass of the rocket and when a new Rocket is used. This method must be overridden to void any cached data necessary. The method must call super.voidMassCache() during its execution.

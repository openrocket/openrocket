Detailed Description

This class allows for a FinSet to be printable. It does so by decorating an existing finset (which will not be modified) and rendering it within a JPanel. The JPanel is not actually visualized on a display, but instead renders it to a print device.

Render the fin set onto the graphics context. This is done by creating a GeneralPath component that follows the outline of the fin set coordinates to create a polygon, which is then drawn onto the graphics context. Through-the-wall fin tabs are supported if they are present.

Detailed Description

Class that defines different cluster configurations available for the InnerTube. The class is immutable, and all the constructors are private. Therefore the only available cluster configurations are those available in the static fields.

List<Double> net.sf.openrocket.rocketcomponent.ClusterConfiguration.getPoints ( )

Returns the relative positions of the cluster components. The list is of length 2*getClusterCount() with (x,y) value pairs. The origin is at (0,0) and the values are positioned so that the closest clusters have distance of 1.

a list of (x,y) coordinate pairs.

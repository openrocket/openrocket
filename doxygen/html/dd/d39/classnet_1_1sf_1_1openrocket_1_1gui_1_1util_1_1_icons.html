§ loadImageIcon()

static ImageIcon net.sf.openrocket.gui.util.Icons.loadImageIcon ( String  file,
String  name 
)
static

Load an ImageIcon from the specified file. The file is obtained as a system resource from the normal classpath. If the file cannot be loaded a bug dialog is opened and null is returned.

Parameters
filethe file to load.
namethe description of the icon.
Returns
the ImageIcon, or null if could not be loaded (after the user closes the dialog)

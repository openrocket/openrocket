§ EXTRA_SCALE

final double net.sf.openrocket.gui.scalefigure.ScaleFigure.EXTRA_SCALE = 1000
static

Extra scaling applied to the figure. The f***ing Java JRE doesn't know how to draw shapes when using very large scaling factors, so this must be manually applied to every single shape used.

The scaling factor used is divided by this value, and every coordinate used in the figures must be multiplied by this factor.

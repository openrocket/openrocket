Detailed Description

An abstract aerodynamic calculator implementation, that offers basic implementation of some methods and methods for cache validation and purging.

§ checkCache()

final void net.sf.openrocket.aerodynamics.AbstractAerodynamicCalculator.checkCache ( Configuration  configuration)
protected

Check the current cache consistency. This method must be called by all methods that may use any cached data before any other operations are performed. If the rocket has changed since the previous call to checkCache(), then voidAerodynamicCache() is called.

This method performs the checking based on the rocket's modification IDs, so that these method may be called from listeners of the rocket itself.

Parameters
configurationthe configuration of the current call

§ voidAerodynamicCache()

void net.sf.openrocket.aerodynamics.AbstractAerodynamicCalculator.voidAerodynamicCache ( )
protected

Void cached aerodynamic data. This method is called whenever a change occurs in the rocket structure that affects the aerodynamics of the rocket and when a new Rocket is set. This method must be overridden to void any cached data necessary. The method must call super.voidAerodynamicCache() during its execution.
